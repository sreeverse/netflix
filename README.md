 Introduction
This project focuses on analyzing a Netflix dataset that contains information about movies and TV shows. The main goal is to explore content trends, compare categories, and generate meaningful insights using data analysis and visualization techniques.

 Dataset Overview
The dataset includes details of Netflix content where each row represents a title. Key columns include:
Show_id – Unique identifier
Type – Movie or TV Show
Title – Name of content
Director – Director name
Cast – Actors involved
Country – Production country
Date_added – Date added to Netflix
Release_year – Year of release
Rating – Age rating (PG, TV-MA, etc.)
Duration – Length (minutes/seasons)
Listed_in – Genre/category
Description – Summary

Objectives
Analyze trends of Movies vs TV Shows over time
Identify top content-producing countries
Find most common genres on Netflix
Study distribution of content ratings
Compare ratings between Movies and TV Shows

 Data Analysis Process
The project follows 5 key steps:
Data Collection
Data Cleaning
Data Exploration (EDA)
Data Analysis
Conclusion

 Data Cleaning Steps
Replaced missing values with "Unknown"
Removed unnecessary or low-quality data
Fixed incorrect values (e.g., swapped rating & duration)
Split duration into time and unit
Converted columns into structured formats
Removed duplicates
Result: Clean, structured, and reliable dataset

 Exploratory Data Analysis (EDA)
 
 Statistical Analysis
Counted total Movies and TV Shows
Analyzed release year distribution
Identified most frequent ratings and genres
Studied overall data patterns

 Visualization
Bar charts for comparisons
Line charts for growth trends
Count plots for ratings and genres
Country-based analysis graphs

 Key Insights
Movies are more common than TV Shows
Content increased significantly after 2015
The United States produces the most content
Peak content production occurred around 2018–2020
Netflix mainly focuses on drama and entertainment genres
Most content targets teens and adults
TV Shows are gradually increasing over time

 Correlation Analysis
Examined relationships between variables
Example: Release Year vs Duration
Used correlation matrix and heatmaps

 Conclusion
Netflix has a large and growing collection of content, with a stronger focus on movies.
 The platform has seen rapid growth in recent years and offers diverse content from multiple countries, mainly targeting mature audiences.
